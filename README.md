# Wazuh SOC Lab

This project demonstrates the setup of a small SOC lab using Wazuh SIEM.

The goal is to understand security monitoring and log analysis.

Tools Used

Wazuh SIEM
Kali Linux
CentOS
VirtualBox

Lab Components

Manager : Wazuh Manager running on CentOS
Attacker : Kali Linux
Endpoint : Windows system
